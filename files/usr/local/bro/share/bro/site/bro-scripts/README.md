# FOXHOUND-NSM 

RaspberryPi 3 NSM system based on [Bro](https://www.bro.org). Suitable for a home 'blackbox' deployment.

### Descriptions
* [geoip] - Geolocation added to the conn.log
* [extract] - Extract files based on module loaded.
